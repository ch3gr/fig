Two dimensional images of any kind, have long been stored and displayed digitally. To do so the image is broken down into a set number of pixels, with a pre-defined color depth. So the digital representation of the image is essentially just a series of color values(1).

If we look at this the other way, we can say that any given digital canvas has a certain number of possible combinations of color values among its pixels(2). And since any kind of image can be digitized, we could also assume that every single respresentation can be found in one of these possible combinations.

The definition of a digital image is proportional to the number of pixels and colors. The possible combinations on the other hand are increasing in a geometric/exponential/logarithmic? way in relation to the number of pixels and colors. This means that the possible combinations of a digital canvas of a substantial resolution, may be an astronomical number, but never the less a finite number.

This simple one button software, gives the ability to the user to iterate through all the color combinations and effectively generate every possible image within a given canvas. No artistic skill is required what so ever, just enough time to go through all these combinations. The grand majority of the results will be non descript(?), but the user will eventually come to see meaninfull images. (Images that resemble photos of known objects, familiar people, or even people that haven't been born yet)

Of course it won't take long for the user to realise that even the lifespan of the universe is not enough to go through all the possible combinations of a canvas with (inequate?) enough definition. Still the software can allow the user to explore all these possibilities and get an understanding of the vastness of such an astronomical numberï¿½.
